# HeapSort String Sorting in Java

This project implements the HeapSort algorithm in Java to sort strings in ascending or descending order.

## File Containing the Required Method
The required method is in:

HeapSort.java

## Required Method Signature
public static void HeapSort(String[] arr, int order)

## Behavior
- order == 0 → ascending
- order != 0 → descending

## Compile
javac HeapSort.java

## Run
java HeapSort
